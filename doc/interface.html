# 接口文档示例

简要描述：

  • 新增标签

请求URL:

  • /tag/addTag

请求方式：

  • post

请求表头：

  • Content-Type: application/json

参数：

参数名 必选 类型 说明
type 是 int 标签类型（1-地区编码）（2-品牌类型）（3-卡券类型）
tagName 是 string 标签名称

参数示例：

{
    "type": 1,
    "tagName": "玉溪"
}

返回示例：

{
    "code": 200,
    "message": "success",
    "result": {
        "id": "T15534139754609694",
        "type": 1,
        "selectType": 1,
        "tagName": "玉溪",
        "status": 2,
        "createTime": "2019-03-24 15:52:55",
        "modifyTime": "2019-03-24 15:52:55",
        "startTime": null,
        "endTime": null
     }
}

返回参数说明：

参数名 类型 说明
id string 标签id
type int 标签类型（1-地区编码）（2-品牌类型）（3-卡券类型）
selectType int 选择类型（1-单选）（2-复选）
tagName string 标签名称
status int 标签状态 （1-草稿）（2-上架）（2-下架）
createTime string 标签创建时间
modifyTime string 标签修改时间
startTime string 标签开始展开时间
endTime string 标签结束展开时间
